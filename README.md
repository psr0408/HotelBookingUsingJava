Room Booking System
Technologies: Java, Servlet, JSP, MySQL, HTML, CSS
Developed a room booking system with both user and admin functionalities using Java, Servlet, and JSP.
Implemented user authentication and registration processes with secure login systems and session management.
Designed and developed 6 dynamic pages including user-login.jsp, admin-login.jsp, user-register.jsp, admin-register.jsp, user-welcome.jsp, and admin-welcome.jsp.
Created a cancellation feature allowing users to cancel bookings through the cancelbooking.jsp page.
Integrated JDBC for database operations and used MySQL for storing user data, room availability, and booking records.
Ensured the system is fully responsive by using HTML and CSS for the frontend design.
