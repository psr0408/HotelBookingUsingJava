Room Booking System is a web application developed using Java, Servlets, JSP, MySQL, HTML, and CSS. The system provides separate functionalities for users and admins, including secure login and registration processes. Users can make and cancel room bookings, while admins can manage the system’s backend. The application consists of key pages such as user-login.jsp, admin-login.jsp, user-register.jsp, admin-register.jsp, user-welcome.jsp, admin-welcome.jsp, and cancelbooking.jsp. The system is fully integrated with JDBC for database operations and MySQL to store user and booking data, ensuring a smooth and responsive user experience.
